# Карл Густав Юнг: Структурний аналіз та інтерпретація життя і творчості

Цей репозиторій містить повний текст монографії **«Карл Густав Юнг: Структурний аналіз та інтерпретація життя і творчості»**, виконаної з використанням системи **SUBIT** (Subjective Identity). У роботі вперше систематизовано весь доробок Карла Юнґа у вигляді 64 архетипних ситуацій, що утворюються перетином трьох кватерних вимірів: **WHO** (суб’єктна позиція: ME, WE, YOU, THEY), **WHERE** (просторова орієнтація: EAST, SOUTH, WEST, NORTH) та **WHEN** (часова фаза: SPRING, SUMMER, AUTUMN, WINTER).

Монографія пропонує:
- Ретроспективний огляд життя та праць Юнґа крізь SUBIT-матрицю.
- Детальний аналіз алхімічних студій Юнґа зі зіставленням 64 операцій «Трактату про камінь» та архетипних ситуацій SUBIT.
- Дослідження синхронії та зв’язку з *І-Цзін* як паралельної системи 64 гексаграм.
- Клінічні імплікації: використання SUBIT у діагностиці, аналізі снів та навігації терапевтичного процесу.
- Перспективи розвитку аналітичної психології в контексті структурного підходу.

## Вступ

Карл Густав Юнґ залишив величезний корпус текстів, який досі не має єдиної систематичної карти. Пропонована монографія заповнює цю прогалину, використовуючи структуру SUBIT, що ґрунтується на юнґівських інтуїціях про кватерність, архетипи та циклічність психічного розвитку. Кожен із 64 архетипів отримує назву, опис, зв’язок із ключовими юнґівськими концептами (его, тінь, аніма/анімус, Самість) та конкретне завдання для індивідуації. Такий підхід дозволяє:

- Побачити єдність розрізнених праць Юнґа – від ранніх психіатричних досліджень до пізніх алхімічних студій.
- Застосувати SUBIT як інструмент аналітичної роботи (аналіз снів, діагностика архетипних ситуацій, планування терапії).
- Встановити міст між юнґіанською традицією та сучасними системними підходами (теорія інформації, когнітивна наука, цифрова гуманітаристика).

## Структура репозиторію

Репозиторій організовано для зручного читання, відтворюваності та автоматизованої збірки. Основні папки:

```
carl-gustav-jung-subit/
├── monograph/               # Повний текст монографії (розбито на розділи)
├── assets/                  # Ілюстрації, таблиці, діаграми
├── build/                   # Сценарії та шаблони для збірки (Pandoc, LaTeX)
├── supplementary/           # Додаткові матеріали (інтерактивний атлас, презентації, дані)
└── docs/                    # Згенерована HTML-версія (GitHub Pages)
```
